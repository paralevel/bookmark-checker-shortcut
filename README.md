# Bookmark Checker (shortcut)

Version 2.0: https://www.icloud.com/shortcuts/ecc2c8b40a77458480d32788f2a44bb6 <sub>[(Version 1.0)](https://www.icloud.com/shortcuts/eb0a09fe9a1141e29bcff987f1b55752)</sub>

Shortcut for iOS and macOS that searches for the current web page in your bookmarks. Finds variant URLs that end/don’t end with an anchor/URI fragment or a “/”, and also URLs with the same title – useful for example when the URL has been changed since the page was bookmarked. On iOS, the shortcut only works from the Share menu in Safari – on macOS, it works both independently and from Safari’s Share menu.

Options
- Add page to bookmarks
- Delete bookmarks
- Open “Bookmarks”
